### 与视频课程的不同点
- express版本不同，教程中使用的是2.x版本，本例使用新版的4.x版本。两个版本具有较大的出入。
- 操作mongdb的package不同，教程使用mongoose，本例使用mongodb，使用上mongoose略简单。
- 数据库文件存储内容有少量不一致。
- 未完成对grunt等自动化工具的使用。
- 本例对登录，录入信息等操作实现Ajax异步操作，但是尚未完成，评论的添加也需改为异步操作。
